## 1. VISÃO GERAL: POR QUE ENERGIA SOLAR?

### Dados do Researcher
- **Margem: 70-150%** (a maior entre todos os nichos)
- **Concorrência: BAIXA** (poucas marcas estabelecidas)
- **Sazonalidade: CRESCENTE** (demanda acelerada em 2025)
- **Ticket médio: R$ 200 - R$ 2.000** (produtos desde R$97 a R$1.497)
- **Produtos**: Painéis portáteis, inversores, kits solares, baterias, carregadores solares
- **Fornecedor principal**: Centrosolar + AliExpress

### Oportunidade
> Enquanto todos focam em acessórios de celular (concorrência ALTA), energia solar tem espaço para crescer com CAC baixo e ROAS alto.


## 4. ESTRATÉGIA DE ESCALA

### 4.1 Fase 1: TESTE (Semanas 1-2)

**Ações:**
- Rodar 3-5 creatives diferentes
- Testar 3 audiências por campanha
- Budget: R$ 150-200/dia total

**Métricas para Decisão:**
| Sinal | Ação |
|---|---|
| CPL < R$ 3 (TOFU) | Escalar +50% |
| ROAS > 2x (BOFU) | Escalar +30% |
| CTR > 2% | Continuar, testar mais audiências |
| CPL > R$ 5 ou ROAS < 1.5x | Pausar campanha |

### 4.2 Fase 2: ESCALA (Semanas 3-6)

**Ações:**
- Duplicar campanhas vencedoras
- Expandir Lookalikes para 3-4%
- Adicionar TikTok Ads
- Budget: R$ 500-1.000/dia total

**Critérios para TikTok:**
```
Se FB/IG: CPL < R$ 3 E ROAS > 3x
          → Escalar para TikTok
Se TikTok: CPL < R$ 2 E video view rate > 40%
           → Escalar budget TikTok para 50% do total
```

### 4.3 Fase 3: DOMINAÇÃO (Semanas 7-12)

**Ações:**
- Concentrar 70% do budget no winning ad set
- Criar campanha de retargeting agressiva
- Lançar novos funis para nichos secundários (LED, Pets)
- Budget: R$ 1.000-3.000/dia total

**Meta de Resultado:**
```
Mês 1: R$ 30.000-50.000 faturamento
Mês 2: R$ 80.000-120.000 faturamento
Mês 3: R$ 150.000-250.000 faturamento
```
