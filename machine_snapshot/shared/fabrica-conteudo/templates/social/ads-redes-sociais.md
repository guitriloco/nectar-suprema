## 📐 ESTRUTURAS DE COPY

### FORMATO 1: PROBLEMA → SOLUÇÃO
```
"Você já [problema]?

A solução é [método].

{Evidência de resultado}.

→ [CTA]"
```

### FORMATO 2: ESTATÍSTICA/ DADO
```
"{X}% das pessoas [ação] porque [motivo].

A solução? [método].

{Evidência}.
→ [CTA]"
```

### FORMATO 3: DEPOIMENTO DIRETO
```
"'{testemunho curto com resultado numérico}'
{Evidência}.
→ [CTA]"
```

### FORMATO 4: PERGUNTA → RESPOSTA
```
"Você quer [benefício]?

Aqui está como: [método].

{Evidência}.
→ [CTA]"
```

### FORMATO 5: URGÊNCIA/ESCASSEZ
```
"⚠️ [Situação de escassez].

[método] disponível por [tempo/condição].

{Evidência}.
→ [CTA]"
```

## 📱 TIKTOK

### HOOKS (3 Primeiros Segundos)
```
"Você está cometendo 3 erros..."
"Em 30 dias eu fiz R$ [X]..."
"Esse fornecedor entregou em [X] dias..."
"O segredo de [nicho] é..."
```

### SCRIPT REELS (15-60s)
```
[0-3s] HOOK: "{Frase que prende atenção>"
[TEXTO]: "Não saia daqui"

[3-15s] PROBLEMA: "{Explicação do problema}"
[TEXTO]: "O problema é..."

[15-30s] SOLUÇÃO: "{Método}>"
[TEXTO]: "A solução é..."

[30-45s] PROVA: "{Depoimento ou dado}"
[TEXTO]: "Funciona assim..."

[45-60s] CTA: "{Chamada para ação}"
[TEXTO]: "Link na bio"
```

## 🎯 MELHORES PRÁTICAS

### IMAGEM/VIDEO
- [ ] Primeiro frame captura atenção
- [ ] Texto legível em mobile
- [ ] Cores contrastantes
- [ ] Face humana (se possível)
- [ ] Call-to-action visível

### COPY
- [ ] Hook nos primeiros 3 segundos
- [ ] Linguagem natural, não robotic
- [ ] Número específicos > números redondos
- [ ] Emojis estratégicos (não excess0)
- [ ] Hashtags relevantes (máx 10)

### TARGETING
- [ ] Audiência definida
- [ ] Interesses específicos
- [ ] Comportamentos relevantes
- [ ] Exclusões claras
