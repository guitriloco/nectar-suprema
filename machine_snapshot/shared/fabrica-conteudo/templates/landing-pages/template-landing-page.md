## 🎯 VARIANTES DE LANDING PAGE

### VARIANTE A: LEARN MORE (Educacional)
```
[HEADLINE] "Aprenda o método completo para [benefício]"

[FOCUS] Explicar método, não vender
[CTA] "Quero Aprender Mais" / "Baixar Guía"
[PRICE] Raramente mostra preço aqui
```

---

### VARIANTE B: LIMITED OFFER (Escassez)
```
[HEADLINE] "{Resultado} ou seu dinheiro de volta"

[COUNTER] "⏰ Oferta termina em: {contagem}"

[CTA] "Garantir Minha Vaga Agora"
[PRICE] Visível com desconto
```

---

### VARIANTE C: SOCIAL PROOF (Validação)
```
[HEADLINE] "{Número}+ pessoas já [resultado]"

[TESTIMONIALS] 3-5 depoimentos grandes
[CTAS] Múltiplos CTAs na página
[PRICE] Médio destaque
```

---

### VARIANTE D: VIDEO CENTRIC (VSL Completo)
```
[HEADLINE] "Assista ao vídeo e descubra..."

[VIDEO] Grande, principal elemento
[COPY] Mínimo ao lado do vídeo
[CTA] "Quero Assistir" / "Ver Detalhes"
```
