## 🎯 REGRAS DE OURO

1. **Assunto é Rei** — Sem clique no assunto, email não é lido
2. **Primeiros 3 segundos** — Capture atenção ou perde o leitor
3. **Um email = Uma ação** — Não sobrecarregue
4. **Valor antes da venda** — Dê antes de pedir
5. **Segmentação** — Diferentes sequências para diferentes perfis
6. **Teste constante** — A/B test subjects, times, content
