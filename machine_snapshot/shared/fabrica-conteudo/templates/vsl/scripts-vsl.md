# TEMPLATE: VSL (VIDEO SALES LETTER)
## Scripts de 15s a 90s para Reels, TikTok, Facebook

---

## 📐 ESTRUTURA VSL 60-90 SEGUNDOS

### CENA 1 — HOOK (0-10s)
```
[CENA]: Close-up do rosto, expressão séria, fundo escuro
[SOM]: Beat drop / Música tensa

"Para de [ação que não funciona]...
Para de [outra ação que não funciona]...

[PAUSA DRAMÁTICA]

...porque tem um nicho que está explodindo no Brasil...
e 90% dos vendedores estão ignorando."

[TEXTO NA TELA]: "{Nicho}"
```

### CENA 2 — PROBLEMA (10-25s)
```
[CENA]: Cortes rápidos, tom intenso, imagens de frustração

"Eu sei o que você está pensando.
'Isso sounds muito bom pra ser verdade.'

[PAUSA]

Eu pensei a mesma coisa. Até testar.

Há {tempo} eu recebi {informação}.
[Testei]. [Resultado rápido]."
```

### CENA 3 — REVELAÇÃO (25-45s)
```
[CENA]: Persona mais calmo, explanatory

"Aí eu percebi algo:
Não era só ter {recurso}. Era ter o {recurso CERTO}.

Eu criei um método: {método}.
[Testar]. [Classificar]. [Usar].

E separei tudo numa lista.

[TEXTO]: "+{Quantidade} {itens}
✓ {Benefício 1}
✓ {Benefício 2}
✓ {Benefício 3}"
```

### CENA 4 — PROVA SOCIAL (40-55s)
```
[CENA]: Screenshots / Comentários / Mensagens

"Não acredita em mim?
Olha o que outras pessoas estão dizendo:

[TEXTO]: "'{testemunho com números}'
— {nome}, {plataforma}"

[TEXTO]: "'{testemunho}'
— {nome}, {plataforma}"

Resultados reais. De pessoas que decidiram agir."
```

### CENA 5 — OFERTA (55-75s)
```
[CENA]: Persona olhando direto para câmera, tom empolgante

"Agora você pode ter acesso a tudo isso.

[TEXTO]: "{Nome do Produto}"
- {Componente 1}
- {Componente 2}
- {Componente 3}

Mas tem uma coisa:
Esse preço é só pra lançamento.

[TEXTO]: "De {preço} por {preço}

E tá disponível por tempo limitado.
Depois volta pro preço cheio."
```

### CENA 6 — FECHAMENTO (75-90s)
```
[CENA]: Close-up, tom pessoal, genuíno

"Escuta...
Se você quer {benefício}...
...você precisa {algo específico}.

Eu tenho.

Clique no link da bio.
Assista o vídeo na página de vendas.
Leia com atenção.

E se decidir entrar, eu vou estar lá do outro lado.

[TEXTO]: "BOA SORTE! 🔥"

Nos vemos do outro lado."
```

## 🎬 VERSÕES CURTAS

### VERSÃO 30s — PARA TRÁFEGO ORGÂNICO

```
[HOOK - 0-3s]
"3 erros que te impedem de lucrar com [nicho]:"
[TEXTO]: "{Nicho} — {Margem}%"

[CORPO - 3-15s]
"1 — Escolher [erro 1]
2 — Não saber [erro 2]
3 — Não ter [erro 3]"

[SOLUÇÃO - 15-25s]
"Eu tenho a solução: {nome do produto}.
{Resumo do diferencial}."

[CTA - 25-30s]
"Salva esse vídeo. Comenta '{palavra-chave}'.
DM eu. Te mando os detalhes."
```

### VERSÃO 15s — PARA STORIES/RETARGETING

```
[HOOK - 0-3s]
"⚠️ O preço da {produto} termina HOJE."
[SOM]: Alarme

[CORPO - 3-10s]
"{benefício}. {componente}. {resultado}.
Tudo por {preço}."

[CTA - 10-15s]
"Arraste pra cima. Garantir preço.
Agora."
```

## 📋 ELEMENTOS VISUAIS

### PRIMEIROS 3 SEGUNDOS
- [ ] Hook visual (contraste, movimento)
- [ ] Texto grande na tela
- [ ] Som/Podcast que prende

### DURANTE VÍDEO
- [ ] Cortes a cada 3-5 segundos
- [ ] Texto reforçando áudio
- [ ] Números em destaque
- [ ] Screenshots de proof

### FECHAMENTO
- [ ] Persona olhando direto
- [ ] Texto com CTA claro
- [ ] Música que motiva

## 📝 NOTAS DE PRODUÇÃO

### Equipamento Mínimo
- Smartphone com câmera boa
- Luz natural ou ring light
- Microfone lapela (opcional)
- Fundo neutro

### Iluminação
- Luz frontal (evitar sombras)
- Fundo com contraste
- Tom de pele natural

### Dicas de Performance
- Olhar para câmera (não para baixo)
- Falar natural, como com amigo
- Pausas dramáticas nos momentos certos
- Energia alta no início e fim
