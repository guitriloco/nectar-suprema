# TEMPLATE: HOOKS EXPLOSIVOS
## Gatilhos de Atenção para Qualquer Plataforma

---

## 🎯 O QUE FAZ UM HOOK EXPLOSIVO?

1. **Provoca** — Faz a pessoa parar e prestar atenção
2. **Identifica** — Mostra que entende o problema
3. **Promete** — Indica uma solução/transformação
4. **Urgencia** — Cria curiosidade para continuar

---

## 📝 ESTRUTURAS DE HOOK

### HOOK 1: PROBLEMA IDENTIFICADO
```
"Você está [descrição situação]...

Mas talvez você não saiba que [fato desconocido]..."
```

**Exemplo Mutante:**
```
"N você está tentando perder peso...

Mas talvez você não saiba que 90% das dietas falham
porque [motivo oculto]..."
```

---

### HOOK 2: CONTRASTE EXPLOSIVO
```
"{Coisa comum} vs. {Coisa incomum}:

A diferença entre [grupo A] e [grupo B]
é [fator decisivo]..."
```

**Exemplo Mutante:**
```
"Vender power bank vs. vender energia solar:

A diferença entre lucrar R$ 100 e R$ 1.000
é [fornecedor certo]..."
```

---

### HOOK 3: ESTATÍSTICA IMPACTANTE
```
"{Número}% das pessoas [ação]...

Porque [motivo]. E a solução é [método]..."
```

**Exemplo Mutante:**
```
"90% dos dropshippers desistem em 3 meses...

Porque [fornecedor errado]. E a solução é
[lista quente validada]..."
```

---

### HOOK 4: PERGUNTA RETÓRICA
```
"Você sabia que [fato surpreendente]?

E o melhor: [benefício]. Sem [objeção comum]..."
```

**Exemplo Mutante:**
```
"Você sabia que existe um nicho com
margem de 150% e baixa concorrência?

E o melhor: não precisa de estoque.
Só precisa de [fornecedor certo]..."
```

---

### HOOK 5: CITAÇÃO/CASE DE SUCESSO
```
"'{Depoimento com resultado específico]'

Isso não é promessa. É resultado de [pessoa real]
que [ação com método]..."
```

**Exemplo Mutante:**
```
"'Vendi R$ 2.300 em uma semana usando
a lista de fornecedores errados.'

Isso não é promessa. É resultado de
Maria, São Paulo, que [método]..."
```

---

### HOOK 6: URGÊNCIA TEMPORAL
```
"⚠️ [Situação]...

E você tem [tempo limitado] para [benefício]..."
```

**Exemplo Mutante:**
```
"⚠️ O preço da lista de fornecedores
termina HOJE...

E você tem 12 horas para garantir
pelo menor preço..."
```

---

### HOOK 7: LISTAGEM/CHECKLIST
```
"3 erros que impedem você de [resultado]:

1. [Erro 1]
2. [Erro 2]
3. [Erro 3]"

E a solução? [Método]..."
```

**Exemplo Mutante:**
```
"3 erros que impedem você de lucrar com dropshipping:

1. Escolher fornecedor errado
2. Não validar tempo de entrega
3. Não ter copy que converte

E a solução? [Lista Quente]..."
```

---

### HOOK 8: PROMISSA TRANSFORMADORA
```
"Imagine [transformação completa]...

Sem precisar [objeção comum].
Só com [método simplificado]..."
```

**Exemplo Mutante:**
```
"Imagine acordar amanhã com notificações
de vendas no seu celular...

Sem precisar comprar estoque.
Só com os fornecedores certos..."
```

---

### HOOK 9: DESMISTIFICADOR
```
"Todo mundo diz que [crença popular]...

Mas a verdade é [realidade diferente]..."
```

**Exemplo Mutante:**
```
"Todo mundo diz que dropshipping
não funciona no Brasil...

Mas a verdade é que funciona sim —
só precisa de [fornecedores certos]..."
```

---

### HOOK 10: SEQUÊNCIA NUMÉRICA
```
"Você precisa de exatamente [número] coisas
para [resultado]:

1. [Coisa 1]
2. [Coisa 2]
3. [Coisa 3]..."
```

**Exemplo Mutante:**
```
"Você precisa de exatamente 3 coisas
para fazer dropshipping funcionar:

1. Fornecedor validado
2. Produto com demanda
3. Copy que converte..."
```

## 🎯 COMO USAR

1. **Escolha** o hook baseado na plataforma e público
2. **Personalize** com dados específicos do nicho
3. **Teste** diferentes hooks com o mesmo conteúdo
4. **Mede** CTR, tempo de visualização, conversões
5. **Escala** o hook que performar melhor
