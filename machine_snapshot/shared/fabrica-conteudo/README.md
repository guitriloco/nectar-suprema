# FABRICA DE COPY MUTANTE
## Sovereign Reality 2.0 — Templates Adaptáveis para IA

---

## 🎯 O QUE É?

A **Fabrica de Copy Mutante** é um sistema de templates que serve como "DNA" para geração automática de copy para qualquer nicho, produto ou plataforma.

A IA pode usar estos templates como base e aplicar mutações específicas para gerar copy personalizado instantaneamente.

## 🚀 COMO USAR

### 1. Selecionar Template
Escolha o template baseado no objetivo:
- **Hook** → G回首 attention
- **VSL** → Venda por vídeo
- **Landing** → Captura ou venda
- **Email** → Nutrição ou lançamento
- **Social** → Anúncios pagos

### 2. Identificar Variáveis
```
{NICHO} → Fitness, Finanças, Pets, etc.
{PROBLEMA} → Dor específica do público
{BENEFÍCIO} → Resultado desejado
{PREÇO} → Valor da oferta
{PRODUTO} → Nome do produto/serviço
```

### 3. Aplicar Mutação
Adaptar o template ao contexto:
- Mudar tom (formal/informal)
- Ajustar linguagem ao público
- Adaptar para plataforma específica
- Personalizar com dados reais

### 4. Testar e Otimizar
- A/B testar variações
- Medir resultados
- Iterar baseado em dados
