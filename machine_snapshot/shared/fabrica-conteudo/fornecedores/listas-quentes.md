## 📊 VALIDAÇÃO DE NICHO

Para validar se um nicho é quente:

1. ✅ Pesquise no Google Trends (gráfico subindo = bom)
2. ✅ Verifique concorrentes (muitos = nicho validado, poucos = oportunidade)
3. ✅ Teste com R$ 50-100 em ads (like/engajamento rápido = bom)
4. ✅ Olhe reviews negativa (o que estão reclamando = oportunidade)
