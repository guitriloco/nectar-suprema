📝 COMO SOLICITAR CONTEÚDO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  PASSO 1: Vá para GitHub Issues → New Issue
  PASSO 2: Escolha o template adequado
  PASSO 3: Preencha com:
    - Nicho/Produto
    - Plataforma (Facebook, TikTok, Instagram, etc)
    - Formato desejado
    - Informações adicionais

  EXEMPLO DE ISSUE:
  ---
  Title: [COPY] - Fitness - Emagrecimento
  
  Nicho: Emagrecimento
  Plataforma: Facebook Ads
  Formato: Copy longo para video
  Tom: Urgente com contagem regressiva
  ---
  
  RESULTADO: Em breve você receberá:
  ✅ Copy completo
  ✅ Headlines
  ✅ CTAs otimizados
  ✅ Variações A/B
