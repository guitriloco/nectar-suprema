## 🚀 Overview

**Nectar Suprema** is the definitive repository merging the most advanced technologies developed across the `guitriloco` organization. It combines Zero-Knowledge Proof (ZKP) preservation, high-performance API structures, recursive logic engines, and a complete content generation factory.

## 🏗️ Architecture

The repository is structured into two main pillars: **Intelligence (src)** and **Output (content)**.

### 🧠 Intelligence (src/)
- **Nexus Core (`/nexus-core`)**: The central data hub and predictive connectivity layer.
- **Hyper Recursion (`/hyper-recursion`)**: High-speed recursive logic engine for deep data transformation.
- **Sovereign Entity (`/sovereign-entity`)**: The primary execution agent, including the `Sovereign_v5` singularity.
- **Mirror Protocol (`/mirror-protocol`)**: Self-reflection and auto-optimization protocol.
- **VVV Vault (`/vvv-vault`)**: Immutable ledger and ZKP verification engine for technical wealth preservation.
- **Yes Scripts (`/yes-scripts`)**: Advanced automation and integration scripts.
- **Nov API (`/nov-api`)**: Optimized FastAPI structure for high-throughput endpoints.

### 🏭 Content Factory (content/)
- **Copias Geradas**: Sales copies, Landing Pages, Reels scripts, and Emails.
- **Materias Primas**: Base assets for content generation.
- **Fornecedores**: Verified vendor lists.

## 📄 Documentation (docs/)
- **Master Strategy**: The blueprint for ecosystem dominance.
- **Scaling Strategies**: E-book on expansion and growth.

Each module contains its own specific instructions. Generally, the system is designed to work with Python 3.10+ and FastAPI.

```bash
# To start the Nexus Core
cd src/nexus-core
uvicorn main:app --port 8000
```
