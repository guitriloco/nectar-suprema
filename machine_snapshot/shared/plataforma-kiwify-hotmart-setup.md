# Configuração de Plataformas de Vendas - E-commerce + Infoprodutos

## Visão Geral do Projeto
Venda do **"Método de Listas Quentes"** (infoproduto de dropshipping) via redes sociais. Estruturação de e-commerce próprio (Shopify/Nuvemshop) integrado com marketplaces e plataformas de infoprodutos.

---

## 1. Plataforma de E-commerce Recomendadas

### 1.1 SHOPIFY (Recomendado #1)
**Ideal para:** Dropshipping profissional, escalabilidade, integração com múltiplos canais.

```
VANTAGENS:
✅ Integração nativa com Oberlo, CJ Dropshipping, AutoDS
✅ Checkout super rápido (Shop Pay)
✅ Integração com TikTok Shop, Facebook/Instagram Shopping
✅ Temas profissionais prontos para conversão
✅ App de análise e marketing nativo
✅ Suporte 24/7 em português

PLANOS:
- Basic Shopify: R$49/mês (básico, suficiente para início)
- Shopify: R$149/mês (profissional)
- Shopify Plus: sob consulta (empresa)

INTEGRAÇÕES ESSENCIAIS:
├── Oberlo (importar produtos AliExpress)
├── Shopify Email (email marketing nativo)
├── Privy/Lemong (pop-ups de captura)
├── PageFly/Page Builder (landing pages)
├── Trackify/PixelGo (Pixels Facebook/TikTok)
└── Nuvemshop Bridge (para migrar depois)
```

### 1.2 NUVEMSHOP (Recomendado #2)
**Ideal para:** Mercado brasileiro, integração com Mercado Pago, melhor suporte local.

```
VANTAGENS:
✅ Integração nativa com Mercado Pago, PicPay, PIX
✅ Checkout transparente com vários métodos BR
✅ Suporte em português (time BR)
✅ Tema adaptados para público brasileiro
✅ Integração com marketplaces (Mercado Livre, Bula)
✅ Plano gratuito (com limitações)

PLANOS:
- Grátis: suficiente para testar
- Profissional: R$79/mês
- Premium: R$199/mês

INTEGRAÇÕES ESSENCIAIS:
├── Correios, Jadlog, Loggi (frete)
├── Mercado Pago (pagamentos)
├── Nuvemshop Email (email marketing)
└── Rastreamento de pedidos
```

### 1.3 TikTok Shop (Crescimento Acelerado)
**Ideal para:** Produtos visuais (LED, acessórios celular, pets) que funcionam bem em vídeo.

```
SETUP TikTok Shop:
1. Criar conta Seller Center em seller.tiktok.com
2. Vincular loja Shopify ou Nuvemshop
3. Cadastrar produtos (atenção às regras de conteúdo)
4. Criar conteúdo ou usar anúncios TikTok

VANTAGENS:
✅ Commissãomínima inicial (1-10%)
✅ Acesso direto a milhões de usuários
✅ Produtos visuais vendem muito bem
✅ LIVE Shopping (vendas ao vivo)
✅ Exposição orgânica mesmo sem anúncios

CUIDADOS:
- Regulamentação de saúde/beleza
- Não vender produtos proibidos
- Seguir guidelines de comunidade
```
