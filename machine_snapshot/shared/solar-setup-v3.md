## 1. Configuração Kiwify - Setup Completo

5. Complete perfil (dados bancários para recebimento)

CONFIGURAÇÕES RECOMENDADAS:
- Nome da loja: [Marca] Solar Store
- URL de checkout: sua-loja.kiwify.com.br

## 3. Setup Shopify - Tutorial Completo

```

3. Cadastre email ou use Google
4. Nome da loja: "[SuaMarca] Solar Store"
5. Preencha dados pessoais
6. Configure loja (rápido)

CONFIGURAÇÕES INICIAIS:
├── Idioma: Português (BR)
├── Moeda: BRL (R$)
├── Endereço: Seu endereço real ou virtual
├── Timezone: America/Sao_Paulo
└── Plano: Basic Shopify (R$49/mês) para começar

```

│   → Criar template de boas-vindas

│   → Configurar pop-up de exit-intent

│   → Criar landing page de captacao

## 4. Setup Nuvemshop - Tutorial Completo

### 4.1 Criar Conta Nuvemshop
```
PASSO A PASSO:
1. Acesse nuvemshop.com.br
2. Clique em "Criar minha loja"
3. Cadastre-se com email
4. Nome da loja: "[SuaMarca] Solar Store"
5. Complete configuração inicial

VANTAGENS NUVEMSHOP:
✅ Integração nativa com Mercado Pago
✅ Parcelamento sem juros (parceiros)
✅ Planos gratuits disponibles
✅ Suporte em português
```

### 4.2 Configurar Meios de Pagamento
```
MERCADOPAGO (OBRIGATÓRIO):
1. Acesse: developer.mercadopago.com.br
2. Crie conta de vendedor
3. Gere credenciais (Client ID + Client Secret)
4. Cole no painel Nuvemshop

CONFIGURAÇÕES:
├── Parcelamento: até 12x
├── À vista: PIX, boleto
├── Taxa mercado pago: ~4% + IOF
└── Recebimento: D+2 a D+30 (configurável)

OUTROS MEIOS:
├── PIX: nativo Nuvemshop
├── Boleto:附带 Mercado Pago
└── Cartão:附 Mercado Pago
```

### 4.3 Configurar Frete
```
OPÇÕES DE FRETE:
├── Correios (PAC, Sedex)
│   → Integrar conta no painel
│
├── Jadlog (alternativa mais barata)
│   → Contratar conta
│
├── Loggi (cidades principais)
│   → Integrar API
│
└── Retire na loja (se aplicável)
    → Para produtos digitais: não aplicável

CONFIGURAÇÃO FRETE PARA DROPSHIPPING:
- Frete fixo: R$15,90 (simular custo real)
- Frete grátis: apenas para pedidos > R$399
- Prazo estimado: 15-30 dias úteis
```
